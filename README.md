    Setup & Run Instructions:

    	Install Node Modules:

    		1.  Client-Side
    			i. 	"cd client"
    			ii.	"npm install"

    		2.  Server-Side
    			i.	"cd ../"
    			ii.	"npm install"

    		3.	Local Run Options:

    			a.	run client & server simultaneously using concurrently
    				i.	"npm run dev"

    			b.	run only client on http://localhost:3000
    				i.	"npm run client"

    			c.	run only server on http://localhost:5000
    				i. "npm run server"
