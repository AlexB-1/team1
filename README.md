    Setup & Run Instructions:

    	1.	Install Node Modules:
    			a.  npm install (client)
    			b.  npm install (server)

    	2.  Local Run Options:
    	    a.	npm run dev
    			    runs client & server simultaneously using concurrently

    	    b.	npm run client
    			    http://localhost:3000

    	    c.	npm run server
    			    http://localhost:5000
